# Global Navigation component
The `GlobalNav` component is comprised of the `TopNav` and `HamburgerMenu` components.

This take in a series of props, as below, to display data and render the correct state, being logged in/out, and displaying the Sunday Times logo or the standard logo.

## Props
<Props of={GlobalNav} />

The `data` prop has the `NavigationData` type, an array of objects with the following data structure;
```
type MenuItem = {
  title: string;
  url: string;
  slug: string;
};

type MenuItemParent = MenuItem & {
  items?: MenuItem[];
};
```
An example of this can  also be found in the 'Knobs' section on  the 'Canvas' tab, where you can customise the data being fed to the GlobalNav component.
