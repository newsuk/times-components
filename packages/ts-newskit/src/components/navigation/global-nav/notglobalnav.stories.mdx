# Global Navigation component
The `GlobalNavWithCustomDrawer` component is comprised of the `TopNav` and `HamburgerMenu` components.

This takes in a series of props, as below, to display data and render the correct state, being logged in/out, and displaying the Sunday Times or the standard logo.

## Props
<Props of={GlobalNavWithCustomDrawer} />

## Code Example
`<GlobalNavWithCustomDrawer
  isLoggedIn={false}
  isSunday={false}
  data={data}
/>
`

An example of the `data` structure can be found in the 'Controls' section on the 'Canvas' tab, where you can customise the data being fed to the GlobalNavWithCustomDrawer component.
