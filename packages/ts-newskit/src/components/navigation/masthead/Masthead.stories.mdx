# Masthead component
The `Masthead` component is used to display the Times and the Sunday Times logos in the header, along with the date.

This takes in a series of props, as below, to display data and render the correct state, what logo and what date to display.

## Props
<Props of={EditionMasthead} />

## Code Example
`<EditionMasthead
  isSunday={false}
  todaysDate={2023-03-02T13:22:39.971Z}
/>`
