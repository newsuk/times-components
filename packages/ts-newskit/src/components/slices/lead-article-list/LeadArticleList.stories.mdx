# LeadArticle component
The `LeadArticle` component is used to display the Header of slices.

This takes in a series of props, as below, to display heading, paragraph, articleType, url and readingTime.

## Props
<Props of={LeadArticle} />

## Code Example
`<LeadArticle heading='Short title of the card describing the main content' paragraph='Short paragraph description of the article, outlining main story and focus.' articleType='Tag' readingTime='Flag'/>`
