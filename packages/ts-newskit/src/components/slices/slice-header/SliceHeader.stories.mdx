# SliceHeader component
The `SliceHeader` component is used to display the Header of slices.

This takes in a series of props, as below, to display title and href.

## Props
<Props of={SliceHeader} />

## Code Example
`<SliceHeader title='Title' href='https://www.thetimes.co.uk/'/>`

## View Component
