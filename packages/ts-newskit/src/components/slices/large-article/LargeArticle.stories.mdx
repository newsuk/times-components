# LargeArticle component
The `LargeArticle` component is used to display the Header of slices.

This takes in a series of props, as below, to display heading , paragraph , tag , flag , url , listData and tagL1.

## Props
<Props of={LargeArticle} />

## Code Example
`<LargeArticle heading='Short title of the card describing the main content' paragraph='Short paragraph description of the article, outlining main story and focus.' tag='Tag' flag='Flag' listData = ['Unordered list item'] tagL1='TAG'/>`
