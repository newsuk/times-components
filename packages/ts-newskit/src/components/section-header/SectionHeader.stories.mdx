# SectionHeader component
The `SectionHeader` component is used to display section titles across the Times website.

This takes in a `title` prop, as below, to display the correct text for the associated section.

## Props
<Props of={SectionHeader} />

## Code Example
`<SectionHeader title='Title' />`
