# SubscribeBanner component
The `SubscribeBanner` component is used to display a list of available challenges.

## Props
<Props of={SubscribeBanner} />

## Code Example
`<SubscribeBanner
  data={ SubscribeBannerHeader, SubscribeBannerSubheadline, url, title }
/>
`

An example of the `data` structure can be found in the 'Controls' section on the 'Canvas' tab, where you can customise the data being fed to the SubscribeBanner component.
