# HintContainer Component

The `HintContainer` component is used to display hints for the daily quiz.

## Props

<Props of={HintContainer} />

### `title` (optional)
The title of the hint button. This prop is optional.

## Code Example

`<HintContainer
  title="Give me a hint"
  hints={hints}
/>`

An example of the `data` structure can be found in the 'Controls' section on the 'Canvas' tab, where you can customise the data being fed to the FinalScreen component.
