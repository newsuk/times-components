# HintDisplay

The `HintDisplay` component is used to display hints.

## Props

<Props of={HintDisplay} />

### `hints` (required)
An array of hint objects to be displayed.

Each hint object should have the following structure:
- `text` (string): The hint text.

### `totalHints` (optional)
Number of total hints

## Code Example

`<HintDisplay
  hints={[
    { text: "This is the first hint." },
    { text: "This is the second hint." },
    { text: "This is the third hint." },
  ]}
  totalHints= 3
/>`

An example of the `data` structure can be found in the 'Controls' section on the 'Canvas' tab, where you can customise the data being fed to the FinalScreen component.
