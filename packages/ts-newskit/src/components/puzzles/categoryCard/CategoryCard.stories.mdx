# CategoryCard component
The `CategoryCard component` component displays recent stories on puzzles page.

This takes in props listed below to display recent story.

## Props
<Props of={CategoryCard} />

## Code Example
`<CategoryCard type={type} url={url} Icon={Icon}>`
