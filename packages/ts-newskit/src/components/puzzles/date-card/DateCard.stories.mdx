# DateCard component
The `DateCard` component is used to display puzzles date cards.

This takes in a `data` and `sectionColour` props, as below, to display the card.

## Props
<Props of={DateCard} />

## Code Example
`<DateCard data={ date, status, gameLevel, url } sectionColour={sectionColour} />`

## View Component
Please click the 'Canvas' tab for a better viewing experience, where you can review and configure with different titles.

  <NewsKitProvider theme={TimesWebLightTheme}>
    <DateCard data={{ date, status, gameLevel, url }} sectionColour={sectionColour} />
  </NewsKitProvider>
);
