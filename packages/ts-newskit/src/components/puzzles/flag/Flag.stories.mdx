# Flag component
The `Flag component` component displays status inside puzzle card.

This takes in a status prop, as below, to display the status of the puzzle game.

## Props
<Props of={PuzzlesFlag} />

## Code Example
`<PuzzlesFlag data={status}></PuzzlesFlag>`
