# StoryCard component
The `StoryCard component` component displays recent stories on puzzles page.

This takes in props listed below to display recent story.

## Props
<Props of={StoryCard} />

## Code Example
`<StoryCard data={image, altText, title, url, category, timeToRead, imgHiddenMobile, hiddenMobile, mobileDivider} />`

## View Component
