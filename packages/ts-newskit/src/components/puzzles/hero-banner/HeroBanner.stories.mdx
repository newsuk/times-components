# HeroBanner component
The `HeroBanner` component is used to display the short details about the Puzzle and to tell the user to subscribe.

## Props
<Props of={HeroBanner} />

## Code Example
`<HeroBanner
  puzzleName="Crossword"
  loginUrl="https://login.thetimes.co.uk?gotoUrl=https://www.thetimes.co.uk/puzzles"
  puzzleType="crossword"
/>
`
