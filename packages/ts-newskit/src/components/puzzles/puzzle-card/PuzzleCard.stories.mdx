# PuzzleCard component
The `PuzzleCard` component is used to display a puzzle card.

This component takes in the following props:

## Props
<Props of={PuzzleCard} />

## Code Example
`<PuzzleCard data={ id, shortIdentifier, title, publishedAt, status, url, image } />`

## View Component
Please click the 'Canvas' tab for a better viewing experience, where you can review and configure with different titles.
