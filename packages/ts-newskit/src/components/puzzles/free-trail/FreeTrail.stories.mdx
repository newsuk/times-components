# FreeTrail component
The `FreeTrail` component is used to display a list of available challenges.

## Props
<Props of={FreeTrail} />

## Code Example
`<FreeTrail
  data={ freeTrailHeader, freeTrailSubheadline, url, title }
/>
`

An example of the `data` structure can be found in the 'Controls' section on the 'Canvas' tab, where you can customise the data being fed to the FreeTrail component.
