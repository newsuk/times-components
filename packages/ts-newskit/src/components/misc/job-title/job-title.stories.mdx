# Job Title component
The `Job Title` component is used to show the author's contractual job title in either a Tooltip (Desktop) or a Popover (Smaller Devices).

This component takes in a single prop `contractualTitle` as below, to display both job titles.

## Props
<Props of={JobTitle} />

## Code Example
`<JobTitle contractualTitle="Senior Correspondent" />`
