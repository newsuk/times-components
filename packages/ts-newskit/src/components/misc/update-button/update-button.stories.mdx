# UpdateButton component
The `UpdateButton` component is used to highlight when there is an update in an article.

The `UpdateButton` component makes use of the `DelayedComponent` which will set the component to mount or unmount given a delay and intial state.

This component takes in a two props, loading to display the loading spinner and display which should be true when an update has happened.

## Props
<Props of={UpdateButton} />

## Code Example
`<UpdateButton loading={loading} display={true} delay={8000} label='New updates'/>`
