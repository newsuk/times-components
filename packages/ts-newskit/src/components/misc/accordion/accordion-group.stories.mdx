# AccordionGroup component
The `AccordionGroup` component is used to show and hide related content.

This component takes in a two props, as below, to display text based content.

## Props
<Props of={AccordionGroup} />

## Code Example
`<AccordionGroup group={[{ header: "text", content: "text" }]} />`
