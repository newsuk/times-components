# ContentBucket1 component
The `ContentBucket1` component is used to display the main story, supplementary articles and journalist quotes.
It is comprised of the `LeadStory`, `SliceHeader`, `ArticleListItem`, and `JournalistQuote` slice components.

This takes in a series of props, as below, to display the part of the slices.

<Props of={ContentBucket1} />

## Code Example
