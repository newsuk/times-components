# SectionBucket component
The `SectionBucket` component is used to display the main story and supplementary articles with a section header.

This takes in a series of props, as below, to display the part of the slices.

## Props
<Props of={SectionBucket} />

## Code Example
`<SectionBucket {...{ articleStackOne, articleStackTwo, articleStackThree, articleStackFour }} />`
