# LeadStory4 component
The `LeadStory4` component is used to display the main story, supplementary articles and comment quotes.
It is comprised of the `leadArticle`, `articlesBelowMainImage`, `isSummaryEnabled` and `articlesRight` slice components.

This takes in a series of props, as below, to display the part of the slices.

## Props
<Props of={LeadStory4} />

## Code Example
`<LeadStory4 {...{ leadArticle,articlesBelowMainImage,articlesRight,isSummaryEnabled }} />`
