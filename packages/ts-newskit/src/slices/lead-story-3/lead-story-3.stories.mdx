# LeadStory3 component
The `LeadStory3` component is used to display the main story, supplementary articles and comment quotes.
It is comprised of the `leadArticle`, `articles`, and `leadArticles` slice components.

This takes in a series of props, as below, to display the part of the slices.

## Props
<Props of={LeadStory3} />

## Code Example
`<LeadStory3 {...{ leadArticle, articles, leadArticles }} />`
