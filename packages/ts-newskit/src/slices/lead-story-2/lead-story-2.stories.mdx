# LeadStory2 component
The `LeadStory2` component is used to display the main story, supplementary articles and comment quotes.
It is comprised of the `leadArticle` and smaller size of `leadArticle` slice components.

This takes in a series of props, as below, to display the part of the slices.

## Props
<Props of={LeadStory2} />

## Code Example
`<LeadStory2 {...{ leadArticle, comments, largeArticles, verticalArticles,horizontalArticles }} />`
