# ContentBucket3 component
The `ContentBucket3` component is used to display the main story, supplementary articles and comment quotes.
It is comprised of the `leadArticleLeft`,`leadArticleRight`, `ArticleListItem`, and `CommentCard` slice components.

This takes in a series of props, as below, to display the part of the slices.

## Props
<Props of={ContentBucket3} />

## Code Example
`<ContentBucket3 {...{ leadArticleLeft, leadArticleRight, comments, articles }} />`
